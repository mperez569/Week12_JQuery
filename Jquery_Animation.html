JQuery FAQ's

What is JQuery?

JQuery is a library of the JavaScript functions that you are most likely to need as you develop a WebSite

Why do we use JQuery?

I use JQuery to explore and learn more about Web Development in JavaScript. Another reason why I use it is because as a class, we work together using it. And finally, It gets you more in less time

What is harder to learn: JQuery or JavaScript?

Both are challenging to learn, but it is harder to learn JQuery because you need to understand how to use JavaScript. JS is a prerequisite to learning JQuery. But once you learn one of them, it'll be easier to learn to other one.
